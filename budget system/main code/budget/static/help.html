使用帮助

一、看看我们能帮你做什么

编制科研课题预算面临一大堆的头疼问题？

● 科研人员不懂财务，财务人员不懂业务，编制出符合业务又财务要求的预算难上加难；

● 课题参与单位多，预算需要分解、平衡、汇总，沟通过程耗费时间；

● 预算科目众多，支出要求和标准有政策性限制，东翻西找查找依据；

● 预算编完还需要对各科目预算进行详细说明，一个都不能差，否则预算面临被砍的命运。

我们的系统主要针对科研预算编制中各位的痛点，为您解决预算编制的烦恼，大大提高效率和精确度。系统主要具有以下功能：

● 分配和协调课题各单位预算

课题牵头单位可以方便的管理课题参与单位的预算，包括分配各参与单位预算、通过邀请码将参与单位编制的预算关联形成课题预算、查看和协调各单位预算进度，轻松汇总输出整个课题预算。

● 全方位的预算编制辅助

1）内置限制性科目的支出标准，不再东找西找各项标准。比如，出国费的各项标准和各币种的汇率已经内置到系统中，您只需要点击选择目的地国家，系统根据相应的住宿、伙食和公杂标准、汇率、人数天数等数据，自动计算出国费用。省事儿！

2）科目编制要点时刻在您旁边；各个科目支出的要求各不相同，难以找到政策文件规定的汇总，别上火，我们把这些都汇总起来显示在桌面上，随时提示！

3）模块化科目预算编制，精确计算个科目支出。系统根据各科目预算的特点和预算编制要求，将每一科目的预算编制模块化了，您只需要输入相应内容即可，精确为您计算各科目支出。

● 完美汇总输出符合要求的课题预算

各参与单位的预算都跟课题关联并编制完成后，该汇总输出整个课题的预算表和预算说明，系统只为您设计了一个按键，点击它！输出预算各种汇总表格和预算说明，完全按照科技计划要求的样式。

二、两分钟了解如何使用？

国家科技计划项目的实施中，课题是最基本最关键的单元，预算的评审等都以课题为单位，所以我们的预算系统也是以课题为单元来设计的。课题预算的编制主体就是课题牵头单位和各参与单位，一般由课题牵头单位与参与单位一起商量课题预算及其分配，然后各单位分头编制各自预算，最后汇总形成课题预算上报。

（一）我是课题牵头单位

➢ 课题牵头单位要做哪几件事？

课题牵头单位在系统里要做如下几件事情：（1）建立课题预算及其基本信息；（2）管理课题各参与单位；（3）分解和协调各单位预算（4）最后，别忘了编制课题牵头单位自己承担任务的预算。

➢ 如何分配课题预算？

1）分解科目预算。科目预算纵向上在各科目上的分解主要是分配总经费、购置设备费、间接经费和绩效支出，您只需要输入课题的总经费和购置设备费，系统会按照科技计划课题间接经费和绩效支出的比例公式自动计算出间接经费和绩效支出，然后将这些科目的预算根据各家任务情况分配到各单位去。

2）在参与单位间分配。在横向上的经费分配主要是将上述四项预算科目分配到课题各参与单位，因为在课题预算的各个科目中，只有这四项预算在课题层面有比例限制，其他科目只要与任务相关，并没有严格比例限制。这四项预算在各参与单位间的分配不需要按照单位预算占课题总预算的比例等比例分配，可以灵活分配。

➢ 如何将各参与单位编制的预算汇总起来

在您添加一个课题参与单位之后，系统都会为这个单位生成一个邀请码，将此邀请码告诉参与单位后，他/她只需要在其单位预算页面里输入这个邀请码就自动关联上了，关联之后您能够看到该单位预算编制的进展和情况。

➢ 如何导出预算？都有哪些内容？

导出预算非常简单！当课题各参与单位都编制完成预算后，您可以一键导出课题预算。根据国家科技计划课题预算书的格式和要求，系统会为您输出要求的各个表格，如课题预算表、购置设备表、测试化验加工费表等，并且会输出完整的预算说明。这些事课题预算书的核心内容。

（二）我是课题参与单位

➢ 课题参与单位要做几件事情？

课题参与单位的主要工作是：（1）根据课题任务建立自己单位在课题中的预算；（2）编制自己单位对应任务的预算；（3）将本单位预算关联到课题预算中。好了，剩下的汇总输出工作由课题牵头单位负责。

➢ 如何编制我的预算？

编制预算特别简单，系统页面您一看就明白，这里就不耽误您时间了，赶紧编起！

➢ 怎样将我的预算关联到课题预算中去？

在您建立本单位预算管理页面，您会看到有一个邀请码的文本框，打电话给课题牵头单位，让他/她把邀请码告诉您，输入点击提交后，您编制的本单位对应课题任务的预算就会自动关联到课题预算中去。
